Software Refactoring Project Repo

This repository contains all the data, scripts, and visualizations related to software refactoring activities for three major projects: Apache Ant, Apache JMeter, and JFreeChart.

The data captures pre-release (W1) and post-release (W2) refactorings for each version, enabling detailed insights into development trends and code evolution.

The repository includes:

  • Raw refactoring data extracted from project snapshots
  • Interactive Chart.js dashboards for visual comparison of W1 vs W2 refactorings
  • Scripts and tools for reproducing charts or extending analysis to new versions

Use this repository to explore trends, compare versions, and understand how code evolves across releases. All visualizations are interactive, downloadable, and responsive, making it easy to share insights.
